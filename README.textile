**scribe-up-shiro-demo** is a web application to test OAuth client support for Shiro based on "buji-oauth extension":https://github.com/bujiio/buji-oauth (and "Scribe UP":https://github.com/leleuj/scribe-up library).

The web app has to be launched on J2E web application server on port 8080 and default racine context (/). The shirodemo.localhost.com should point to localhost.
